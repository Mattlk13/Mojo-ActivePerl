NAME

Mojo::HelloWorld - Hello World!

SYNOPSIS

  use Mojo::HelloWorld;
  
  my $hello = Mojo::HelloWorld->new;
  $hello->start;

DESCRIPTION

Mojo::HelloWorld is the default Mojolicious application, used mostly for testing.
